Link: https://jogo-da-velha-sable-one.vercel.app/

<h3>Tecnologias usadas:</h3>

- HTML
- CSS
- JavaScript

<h3>Conhecimentos treinados:</h3>

- Marcação e estilização do site
- Responsividade
- Manipulação do DOM usando JavaScript
- Lógica de programação
